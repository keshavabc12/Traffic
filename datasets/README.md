# Bangalore Traffic Dataset

## Download Instructions

1. **Visit Kaggle Dataset Page:**
   - Go to: https://www.kaggle.com/datasets/ravirajsinh45/real-time-traffic-data-bangalore
   - Or search for "Bangalore traffic" on Kaggle

2. **Download the Dataset:**
   - Click the "Download" button
   - You may need to sign in to Kaggle (free account)
   - Extract the ZIP file

3. **Place Dataset Here:**
   - Copy the CSV file(s) to this `datasets/` directory
   - Expected filename: `bangalore_traffic.csv` or similar

## Expected Dataset Schema

The dataset should contain columns like:
- `timestamp` or `date_time` - Date and time of measurement
- `location` or `area` or `road_name` - Location identifier
- `traffic_volume` or `vehicle_count` - Number of vehicles
- `average_speed` - Average vehicle speed (km/h)
- `congestion_level` - Traffic congestion level (1-5 or similar)

## Alternative Datasets

If the above dataset is not available, you can use any Bangalore traffic dataset with similar fields:
- Bangalore's Traffic Pulse
- Bangalore City Traffic Dataset
- Any CSV with traffic metrics for Bangalore locations
